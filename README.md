Hello! My name is Aubrey Johnson, and I am a college student studying electrical engineering.
I started building guitar pedals to help deepen my understanding of my courses (I'm going through Circuits II right now,
so working with this circuit was very helpful). 

This is a heavily distorted Octave Up pedal, meaning that you will hear the note that your guitar is playing doubled and 
up one octave. 

Everything you need to construct this pedal yourself should be included in the files. The only things I am NOT going to include
in this repository are the drilling hole sizes for the pedal enclosure. You should find the following items throughout the 
repository:
-LTSPICE simulation of the circuit

-KiCad files of both the schematic and the PCB

-Source files so you can see where I found the circuit design

-List of components/parts necessary for the project

-Wiring schematic(s) for the peripherals (3PDT switch, potentiometer, power supply port, etc.)

This is only my second time building a pedal, so my PCB design is rather amateur-looking.
Please feel free to tweak any of these files to your liking.

Additionally, if any of the listed items are missing from the repository, they will be added as soon as the pedal is complete
(This project is still a work-in-progress as of the time of writing).
